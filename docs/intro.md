# Guides

This section is for evergreen procedures, reference material, and reusable walkthroughs that should be easier to find than a date-based blog post.

Good candidates include:

- PowerShell setup and troubleshooting procedures.
- Microsoft 365 and Microsoft Graph security reference notes.
- Repeatable operational checklists.
- Longer tutorials that may be revised over time.

Blog posts remain the right place for time-sensitive announcements, research notes, and opinion pieces.
